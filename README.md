# My GitHub Pages Hub

## 概要・説明
**My GitHub Pages Hub** は、自身が作成した、またはよく閲覧する「GitHub Pages」を一つの場所で一元管理・整理できるWebアプリケーションです。
HTMLファイル単体で動作するフロントエンド完結型のシングルページアプリケーション（SPA）でありながら、ローカルデータベース（IndexedDB）やWeb Worker、PWAを活用し、高速かつネイティブアプリのような使用感を実現しています。

さらに、GitHubのPersonal Access Token（PAT）を設定することで、自身のアカウントからPagesが有効なリポジトリを**自動検出**し、最新の更新日時や説明文をAPI経由で**一括同期**することが可能です。

## 主な機能一覧

- **ページの登録・管理**
  - GitHub PagesのURL、タイトル、自由なメモ・タグを登録。
  - カスタムタイトルの設定、またはGitHub APIからの自動タイトル取得。
- **リポジトリの自動検出・一括同期**
  - GitHub PATを利用し、自身の公開/プライベートリポジトリからPagesを自動検出して一括追加。
  - APIを利用して最終更新日やリポジトリの説明文をバックグラウンド（Web Worker）で同期。同期中のスリープ防止機能（Wake Lock API）も搭載。
- **検索とフィルタリング**
  - 登録したページをキーワード（URL、タイトル、メモ）でインクリメンタル検索。
  - よく使うページを「お気に入り（☆）」に登録し、ワンタップで絞り込み。
- **強力なバックアップ・データ管理**
  - **自動バックアップ**: データの変更ごとに最大5世代前まで自動保存され、いつでも復元可能。
  - **インポート/エクスポート**: 全データをJSON形式でファイルとして出力・読み込み。
- **PWA（Progressive Web App）対応**
  - スマートフォンやPCのブラウザから「ホーム画面に追加（インストール）」することで、独立したアプリとして利用可能。

## 使用技術・ライブラリ

本プロジェクトはビルドツールなどを必要とせず、ブラウザの標準機能を最大限に活用して構築されています。

- **フロントエンド**
  - HTML5 / CSS3 / Vanilla JavaScript
  - [Tailwind CSS](https://tailwindcss.com/) (CDN) - スタイリング・UIデザイン
  - [Font Awesome 6](https://fontawesome.com/) (CDN) - アイコン描画
- **ブラウザAPI・機能**
  - **IndexedDB**: データの永続化・ローカル保存
  - **Web Workers API**: APIリクエストなどの重い処理を非同期で行い、UIのフリーズを防止
  - **Wake Lock API**: 一括同期中の画面消灯（スリープ）を防止
  - **PWA (Manifest Injection)**: 動的なマニフェスト生成によるインストール対応
- **外部API**
  - **GitHub REST API**: メタデータ（タイトル・説明・更新日）の取得、リポジトリの検索

## セットアップ・使い方

サーバーの構築やデータベースの用意は不要です。

### 1. アプリの起動
提供されたHTMLファイルをブラウザで開くか、任意のWebサーバー（GitHub Pages等）にホスティングしてアクセスするだけで即座に利用を開始できます。

### 2. GitHub PATの設定（推奨）
APIのレート制限を解除し、自動検出機能を利用するためにGitHubのPersonal Access Token（PAT）を登録することをおすすめします。
1. 画面右上の **「歯車（設定）」アイコン** をクリックします。
2. GitHubで生成したPAT（`repo`等の権限を付与したもの）を入力し、「保存」をクリックします。
*(※PATはブラウザのローカルストレージにのみ安全に保存されます)*

### 3. ページの追加と自動検出
- **手動で追加**: 画面右下の **「＋」ボタン** をクリックし、GitHub PagesのURL（例: `https://username.github.io/repo/`）を入力して保存します。
- **自動検出**: 画面下部のナビゲーションから **「一括同期」** をクリックすると、設定したPATをもとに自身のアカウントをスキャンし、Pagesが存在するリポジトリを自動的に一覧に追加します。

### 4. データのバックアップと復元
- **自動保存からの復元**: 画面下部の **「履歴/復元」** をクリックすると、直近5回分の操作履歴から任意の時点にデータをロールバックできます。
- **ファイルへの書き出し**: 設定画面の **「エクスポート」** からJSONファイルをダウンロードして手元に保管でき、別端末へ移行する際は **「インポート」** から読み込むことができます。
